# LasAduaneras
Buenos días / tardes / noches.

Este repositorio es público para que cualquier personas que quiera verlo, pueda analizar a profundidad que es lo que estamos haciendo, como lo estamos haciendo, y de que maneras.

Gracias a esta API que creamos entre el grupo, podemos enviar y recibir información hacia o desde la base de datos alojada en Microsoft Azure, esto con la finlidad de no tener nuestro servicio alojado unicamente en un servicio, teniendo siempre un soporte sobre donde puede estar cualqueir fallo.

El repositorio es público y está libre para que cualquier persona que quiera copiarlo sea libre de hacerlo, aunque, está hecho para la conexión a nuestra base de datos personal.

Por su atención muchas gracias.
